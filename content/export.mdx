# Export System

Save and load trained models with the .lle format.

## .lle Format

The .lle format is train-lle's native model serialization format. It stores:

- Model architecture
- Trained weights and biases
- Training metadata
- Optimizer state

## Saving Models

```typescript
// Save a trained model
await model.save('my-model.lle');
```

## Loading Models

```typescript
// Load a saved model
const model = await Model.load('my-model.lle');

// Use the loaded model for predictions
const predictions = model.predict(testData);
```

## Export Examples

### Save & Load Workflow

```typescript
import { Model, Trainer, Dataset } from 'train-lle';

// Train a model
const model = new Model({ layers: [...] });
const dataset = new Dataset('training-data.csv');
const trainer = new Trainer(model, dataset);
await trainer.train({ epochs: 100 });

// Save the trained model
await model.save('trained-model.lle');

// Later, in another session...
const loadedModel = await Model.load('trained-model.lle');
const predictions = loadedModel.predict(newData);
```

### Model Inspection

```typescript
const model = await Model.load('model.lle');
console.log(model.summary());
console.log(model.getWeights());
```

> **Tip**: Always save your models after training to avoid losing progress!
