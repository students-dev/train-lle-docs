# Model Types

train-lle supports various neural network architectures.

## MLP (Dense)

Multi-Layer Perceptron with fully connected layers.

```typescript
const model = new Model({
  layers: [
    { type: 'dense', units: 128, activation: 'relu', inputShape: [784] },
    { type: 'dense', units: 64, activation: 'relu' },
    { type: 'dense', units: 10, activation: 'softmax' }
  ]
});
```

Best for: Tabular data, classification, regression.

## CNN

Convolutional Neural Networks for image processing.

```typescript
const model = new Model({
  layers: [
    { type: 'conv2d', filters: 32, kernelSize: [3, 3], activation: 'relu', inputShape: [28, 28, 1] },
    { type: 'maxpool2d', poolSize: [2, 2] },
    { type: 'conv2d', filters: 64, kernelSize: [3, 3], activation: 'relu' },
    { type: 'maxpool2d', poolSize: [2, 2] },
    { type: 'flatten' },
    { type: 'dense', units: 128, activation: 'relu' },
    { type: 'dense', units: 10, activation: 'softmax' }
  ]
});
```

Best for: Image classification, object detection.

## RNN

Recurrent Neural Networks for sequence data.

```typescript
const model = new Model({
  layers: [
    { type: 'lstm', units: 128, inputShape: [null, 10] },
    { type: 'dense', units: 1, activation: 'linear' }
  ]
});
```

Best for: Time series, text processing, sequence prediction.

> **Note**: RNN layers support variable-length sequences with `null` in the input shape.
