# Roadmap

Future development plans for train-lle.

## v0.1.0 (Current)

- ✅ Basic neural network implementation
- ✅ Dense, Conv2D, and LSTM layers
- ✅ CSV and JSON dataset support
- ✅ Model save/load functionality
- ✅ CLI tools
- ✅ Node.js and browser support

## v0.2.0 (GPU Support)

- WebGL acceleration for browser
- CUDA support for Node.js
- Performance optimizations
- Larger model support

## v0.5.0 (WASM)

- WebAssembly compilation
- Cross-platform performance
- Reduced bundle sizes
- Offline PWA support

## v1.0.0 (Transformers)

- Transformer architecture
- Pre-trained models
- NLP capabilities
- Vision transformers
- Hugging Face integration

## Future Plans

- Audio processing pipelines
- Video analysis tools
- Federated learning
- Model compression
- AutoML features
