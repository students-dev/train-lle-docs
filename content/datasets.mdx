# Dataset Formats

train-lle supports multiple data formats for flexibility.

## CSV

Comma-separated values for tabular data.

```csv
feature1,feature2,target
1.0,2.0,0
3.0,4.0,1
5.0,6.0,0
```

```typescript
const dataset = new Dataset('data.csv', {
  hasHeader: true,
  targetColumn: 'target'
});
```

## JSON

JavaScript Object Notation for structured data.

```json
[
  {"input": [1, 2, 3], "output": [0]},
  {"input": [4, 5, 6], "output": [1]},
  {"input": [7, 8, 9], "output": [0]}
]
```

```typescript
const dataset = new Dataset(dataArray);
```

## Images

Directory-based image datasets.

```
images/
├── class1/
│   ├── image1.jpg
│   └── image2.jpg
└── class2/
    ├── image3.jpg
    └── image4.jpg
```

```typescript
const dataset = new Dataset('images/', {
  type: 'image',
  imageSize: [224, 224]
});
```
