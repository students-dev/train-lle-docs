# Installing train-lle

Once Node.js is installed, you can install train-lle using npm.

## Global Installation

For CLI usage:

```bash
npm install -g train-lle
```

## Project Installation

For use in your projects:

```bash
npm install train-lle
```

## Verify Installation

```bash
# Check if installed globally
train-lle --version

# Or in project
npx train-lle --version
```

## Development Installation

For contributing to train-lle:

```bash
git clone https://github.com/students-dev/train-lle.git
cd train-lle
npm install
npm run build
```
